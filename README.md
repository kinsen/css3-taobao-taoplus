## Buid

+ node
+ gulp
+ [LiveReload plugin](https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei)

```bash
$ cd css3-taobao-taoplus

$ npm install

$ gulp
```

Open your browser and go to http://127.0.0.1:8000
